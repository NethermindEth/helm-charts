### Extra Ports

This will create both service and container ports configuration.

```yaml
service:
  extraPorts:
    - name: ws
      port: 1234
      protocol: TCP
```

### Environment Variables

#### From ConfigMap

Mounting a ConfigMap as environment variables is the simplest way to use a ConfigMap in a Pod.

```yaml
envFrom:
  - configMapRef:
      name: '{{ include "generic-app.fullname" . }}'

configMap:
  enabled: true
  data:
    VAR_1: value1
    VAR_2: value2
```

#### From env

```yaml
env:
  - name: MY_ENV_VAR
    value: my-env-var-value
```
